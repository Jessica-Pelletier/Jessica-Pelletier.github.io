About Me

Hi, I’m Jessica Pelletier—a web and mobile developer based in Campbell River, BC.

I recently graduated from North Island College with a diploma in Web and Mobile Application Development. I began the program with the goal of learning how to create websites and design engaging online experiences, and along the way, I discovered a true passion for connecting front-end interfaces to back-end systems to build robust, user-focused applications.

With a love for full-stack development and a focus on user experience, I strive to create digital solutions that not only meet modern web and app standards but also make life easier and more enjoyable for the people who use them.

I’m always excited to explore new tools, tackle creative challenges, and collaborate on projects that push the boundaries of what’s possible online.

Projects

The Impact Atlas

Impact Atlas is a concept prototype for an interactive resource hub mapping social impact initiatives worldwide.

Using Leaflet.js visualization, it demonstrates how to connect changemakers with methodologies and implementation guides through an accessible digital platform.

Time Track

The TimeSheet App is a mobile-first solution built with React Native and Firebase that simplifies time tracking for both employees and employers. It offers features like timesheet management, project tracking, role-based access control, and CSV export for seamless payroll and reporting integration.

GrowCast

An experimental website built with JavaScript, HTML, and CSS on a Node.js backend. It uses the user's location to fetch real-time weather data and garden metrics from the Open-Meteo API.
